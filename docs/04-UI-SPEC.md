# UI 设计规范

## 整体风格

- **设计语言**：现代、简约、玻璃拟态（Glassmorphism）局部点缀
- **视觉调性**：专业工具感，不花哨但有质感
- **中文优先**：界面文案全部中文，字体优先思源黑体 / 系统默认

## 核心组件规范

### SiteCard — 站点卡片

```
┌──────────────────────────────┐
│  ● 状态灯   [分类色条]       │
│                              │
│  [Favicon]  站点名称         │
│             https://xxx.com  │
│                              │
│  站点描述文字最多两行...     │
│                              │
│  ┌──────┐ ┌──────┐          │
│  │ 标签1 │ │ 标签2 │          │
│  └──────┘ └──────┘          │
│                              │
│  ─────────────────────────── │
│  🕐 2分钟前检测  👁 128次    │
└──────────────────────────────┘
```

- 左上角：状态指示灯（绿/红/灰圆点，8px）
- 右上角：分类色条（4px 高，分类颜色）
- 悬浮效果：轻微上移 (translateY -2px) + 阴影加深
- 点击行为：新标签页打开 URL
- 右键/更多按钮：编辑、删除选项

### CategoryFilter — 类型筛选栏

- 水平排列的 Chip/Tab 组件
- "全部" 始终在首位
- 选中态：主色背景 + 白色文字
- 未选中态：透明背景 + 次要文字色
- 每个分类前显示对应 Lucide 图标
- 溢出时水平滚动

### SiteForm — 新增/编辑表单（Modal）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 站点名称 | 文本输入 | 是 | placeholder: "例如：GitHub" |
| 站点地址 | URL 输入 | 是 | placeholder: "https://github.com" |
| 所属分类 | 下拉选择 | 是 | 从分类列表选择 |
| 站点描述 | 多行文本 | 否 | 最多 500 字 |
| 标签 | 标签输入 | 否 | 回车添加，最多 10 个 |

- 模态框居中，背景遮罩
- 底部：取消 + 确认按钮
- 表单校验：实时提示错误

### Header — 顶部导航

- 左侧：项目 Logo + 名称 "Site Manager"
- 中间：搜索栏（带搜索图标，支持 Cmd/Ctrl+K 快捷键聚焦）
- 右侧：主题切换按钮 + 新增站点按钮

### ConfirmDialog — 确认弹窗

- 用于删除操作的二次确认
- 标题："确认删除"
- 内容："确定要删除「{站点名}」吗？此操作不可撤销。"
- 按钮：取消（灰色）+ 确认删除（红色）

## 动效规范

| 场景 | 动效 | 时长 |
|------|------|------|
| 卡片悬浮 | translateY + shadow | 200ms ease |
| 模态框出现 | scale(0.95→1) + fade | 200ms ease-out |
| 模态框关闭 | scale(1→0.95) + fade | 150ms ease-in |
| 筛选切换 | 卡片淡入淡出 | 150ms |
| 状态灯变化 | 颜色过渡 | 300ms |
| 主题切换 | 全局颜色过渡 | 200ms |

## 响应式断点

| 断点 | 宽度 | 卡片列数 | 布局调整 |
|------|------|----------|----------|
| 桌面 | ≥1280px | 4 列 | 完整布局 |
| 小桌面 | ≥1024px | 3 列 | 搜索栏缩短 |
| 平板 | ≥768px | 2 列 | Header 简化 |
| 手机 | <768px | 1 列 | 筛选栏改为下拉 |
